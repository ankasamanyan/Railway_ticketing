# Railway Ticketing app

## How to run the project
• 1 • 
```bash
git clone https://github.com/ankasamanyan/Railway_ticketing.git

```
• 2 • 
```bash
cd Railway_ticketing

```
• 3 •
```bash
make build

```

• 4 •

open http://localhost:4200
